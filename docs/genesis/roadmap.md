# Genesis Engine — Roadmap

## Current State (v1.0.0 Draft)

The Genesis Engine documentation repository is organized as an Enterprise
Knowledge Architecture with 30+ top-level directories. The foundational layer
(constitutions + ontology) is complete. The agent layer (27 agent specs) is
complete. The specification layer (15 specs) covers the core formats and
protocols. The workflow layer (4 workflows) covers the main execution paths.
