Emotional Withdrawal Video Generation Project

Project Overview

This project focuses on building a video generation pipeline specifically tailored for psychology content, with "Emotional Withdrawal" as the primary topic. The system leverages AI to create cinematic videos that explain psychological concepts in an engaging, visual format.

Status: In Progress

Technical Architecture

Core Pipeline

The system uses a multi-stage AI pipeline:

  • Story Generation: Creates structured narratives with emotional arcs
  • Scene Decomposition: Breaks stories into cinematic scenes
  • Dialogue Generation: Creates natural spoken dialogues for each scene
  • Cinematic Prompt Generation: Produces detailed visual prompts for video generation

Technology Stack

The project combines:

  • Frontend: Next.js with TypeScript and Tailwind CSS
  • Backend: FastAPI for API services
  • LLMs: Ollama with qwen2.5:32b and deepseek-coder-v2 models
  • Video Generation: Stable Video Diffusion (SVD) or similar
  • Local Processing: All AI processing happens locally

Psychology Content Focus

The system is specifically optimized for:

  • Emotional Withdrawal psychology concept
  • Clinical and educational video content
  • Visual storytelling for mental health topics
  • Accessible, non-stigmatizing presentation of psychological concepts

Current Implementation

Core Pipeline Components

The system currently includes:

  • Python-based text generation pipeline with dual-model architecture
  • Ollama integration for local LLM processing (qwen2.5:32b and deepseek-coder-v2)
  • Research integration for grounding psychological concepts
  • Structured output formats (YAML) for all pipeline stages

Video Generation Pipeline

The video generation process:

  1. Accepts psychological topic input (e.g., "Emotional Withdrawal")
  2. Generates structured story with emotional arc
  3. Breaks story into cinematic scenes with camera and lighting details
  4. Creates dialogues for each scene
  5. Generates cinematic prompts for video generation models
  6. Assembles final video clips with optional narration
